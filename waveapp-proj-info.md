# Waves of Cox's Bazar Mobile Application

Mobile application for Cox's Bazar Tourism powred by Cox's Bazar DC Office.

## Built with
* **Java**
* **php**
* **Rest-Api**
* **Javascript**
* **MySQL**
* **Firebase Cloud Messaging**
* **Realtime-DB**
* **Google Map Api**
* **GPS Sensor**
* **git**
